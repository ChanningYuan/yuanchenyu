# 项目计划：使用 Hugo PaperMod 构建个人博客

## 1. 项目理解

**这个项目是什么？**
这是一个 **Hugo 主题**，用于构建静态博客网站。

* **核心特点**：极速（无多余 JS/CSS）、响应式设计、SEO 友好、支持深色/浅色模式。

* **适用场景**：个人博客、技术文档、个人作品集。

**能做到什么程度？**
我们可以实现一个专业级的个人博客，包含以下功能：

* **主页模式**：

  * **Regular Mode**：传统博客列表。

  * **Home-Info Mode**：左侧简介，右侧文章列表。

  * **Profile Mode**：全屏个人主页（适合作为 Landing Page）。

* **内容组织**：

  * **归档 (Archives)**：按时间展示所有文章。

  * **搜索 (Search)**：极速客户端搜索 (Fuse.js)。

  * **标签/分类**：自动聚合。

* **阅读体验**：代码高亮（带复制按钮）、文章目录 (ToC)、封面图、多语言支持。

* **互动**：评论系统、社交分享。

## 2. 构建流程 (标准步骤)

1. **环境准备**：安装 Hugo (`brew install hugo`)。
2. **创建站点**：`hugo new site my-blog`。
3. **安装主题**：克隆 `hugo-PaperMod` 到 `themes/`。
4. **配置**：修改 `config.yml`，启用功能。
5. **创作**：编写 Markdown 文章。
6. **发布**：推送到 GitHub Pages / Vercel。
