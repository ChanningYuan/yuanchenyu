# 检查清单 (Checklist)

- [x] 站点标题、描述、作者等元数据已正确配置并显示。
- [x] 个人主页模式 (Profile Mode) 正常工作（头像、标题、副标题、社交图标）。
- [x] 导航菜单链接（首页、归档、分类、标签、搜索）跳转正确。
- [x] 搜索功能可返回相关结果，并支持关键词高亮。
- [x] 归档页面 (Archives) 能按年份正确分组并展示文章。
- [x] 文章详情页正确渲染 Markdown 内容、目录 (ToC)、代码块（带复制按钮）及封面图。
- [x] 主题切换功能（浅色/深色/自动）工作正常，并能记住用户选择。
- [x] 评论系统已配置占位符或实际集成（如 Giscus）。
- [x] 文章底部的社交分享按钮可见并可用。
- [x] 统计分析代码（如 Google Analytics）已在配置中预留。
